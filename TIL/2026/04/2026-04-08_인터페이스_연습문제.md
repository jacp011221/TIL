# 8-7.

```java
// DataAccessObject
package ch08.sec00.test07;

public interface DataAccessObject {
    void select();
    void insert();
    void update();
    void delete();
}
```

```java
// OracleDao
package ch08.sec00.test07;

public class OracleDao implements DataAccessObject {
    private static String name = "Oracle DB";

    @Override
    public void select() {
        System.out.println(name + "에서 검색");
    }

    @Override
    public void insert() {
        System.out.println(name + "에 삽입");
    }

    @Override
    public void update() {
        System.out.println(name + "를 수정");
    }

    @Override
    public void delete() {
        System.out.println(name + "에서 삭제");
    }
}
```
