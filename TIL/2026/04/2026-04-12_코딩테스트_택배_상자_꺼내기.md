### before code

```java
class Solution {
    public int solution(int n, int w, int num) {
        int answer = 0;
        int[] box = new int[w];

        int allRow;
        int topCol;
        if(n%w == 0) {
            allRow = n / w;
            topCol = w;
        } else {
            allRow = n / w + 1;
            topCol = n % w;
        }

        if(allRow%2 == 0) {
            for(int i=0; i<(w - topCol); i++) {
                box[i] = allRow - 1;
            }
            for(int i=(w - topCol); i<w; i++) {
                box[i] = allRow;
            }
        } else {
            for(int i=0; i<(topCol); i++) {
                box[i] = allRow;
            }
            for(int i=(topCol); i<w; i++) {
                box[i] = allRow - 1;
            }
        }

        int numRow;
        int numCol;
        if(num%w == 0) {
            numRow = num / w;
            numCol = w;
        } else {
            numRow = num / w + 1;
            numCol = num % w;
        }

        if(numRow%2 == 0) {
            answer = box[w - numCol] - numRow + 1;
        } else {
            answer = box[numCol - 1] - numRow + 1;
        }

        return answer;
    }
}
```

### after code

```java
class Solution {
    public int solution(int n, int w, int num) {
        int answer = 0;
        int[] box = new int[w];

        int allRow = locationRow(n, w);
        int topCol = locationCol(n, w);

        if(allRow%2 == 0) {
            for(int i=0; i<(w - topCol); i++) {
                box[i] = allRow - 1;
            }
            for(int i=(w - topCol); i<w; i++) {
                box[i] = allRow;
            }
        } else {
            for(int i=0; i<(topCol); i++) {
                box[i] = allRow;
            }
            for(int i=(topCol); i<w; i++) {
                box[i] = allRow - 1;
            }
        }

        int numRow = locationRow(num, w);
        int numCol = locationCol(num, w);

        if(numRow%2 == 0) {
            answer = box[w - numCol] - numRow + 1;
        } else {
            answer = box[numCol - 1] - numRow + 1;
        }

        return answer;
    }

    public static int locationRow(int count, int w) {
        return (count - 1) / w + 1;
    }

    public static int locationCol(int count, int w) {
        return (count - 1) % w + 1;
    }
}
```

- 행과 열을 계산하는 로직이 반복되므로, 이를 static 메소드로 분리해 중복 코드를 줄였다.
- 1-based 인덱스를 0-based 인덱스로 변환한 뒤 계산하는 (count - 1) / w + 1 공식을 사용해 조건문 없이도 동일한 결과를 얻을 수 있게 변경했다.
