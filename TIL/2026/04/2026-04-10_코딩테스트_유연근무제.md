### before code

```java
class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        int answer = 0;
        int[] check = new int[schedules.length];

        for(int i=0; i<schedules.length; i++) {
            if( schedules[i]%100 >= 50 ) {
                schedules[i] = schedules[i] + 50;
            } else {
                schedules[i] = schedules[i] + 10;
            }
        }

        for(int i=0; i<schedules.length; i++) {
            for(int k=0; k<6-startday; k++) {
                if( schedules[i] >= timelogs[i][k] ) {
                    check[i]++;
                }
            }
            for(int k=0; k<startday-1 && k<5; k++) {
                if( schedules[i] >= timelogs[i][8-startday+k] ) {
                    check[i]++;
                }
            }
            if(check[i] == 5) {
                answer++;
            }
        }

        return answer;
    }
}
```

### after code

```java
class Solution {
    public int solution(int[] schedules, int[][] timelogs, int startday) {
        int answer = 0;
        //int[] check = new int[schedules.length];

        for(int i=0; i<schedules.length; i++) {
            if( schedules[i]%100 >= 50 ) {
                schedules[i] = schedules[i] + 50;
            } else {
                schedules[i] = schedules[i] + 10;
            }
        }

        for(int i=0; i<schedules.length; i++) {
            boolean check = true;
            
            for(int k=0; k<6-startday; k++) {
                if( !( schedules[i] >= timelogs[i][k] ) ) {
                    check = false;
                    break;
                }
            }
            for(int k=0; k<startday-1 && k<5; k++) {
                if( !check ) break;
                if( !( schedules[i] >= timelogs[i][8-startday+k] ) ) {
                    check = false;
                    break;
                }
            }
            if(check) {
                answer++;
            }
        }

        return answer;
    }
}
```

- check를 int 타입 배열에서 boolean 타입 로컬 변수로 변경했다.
- 조건식이 한 번이라도 false가 되면 해당 answer은 확정적으로 유지된다. 따라서 이 경우 check를 false로 변경하고, for문을 빠져나가 불필요한 반복을 줄였다.
